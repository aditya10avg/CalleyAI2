# Calley AI

**Calley AI** is an automated cold calling tool designed to simplify and streamline your cold calling efforts. It helps you make multiple calls easily, improving your productivity and efficiency.

## Features
- Automatic dialing
- Call scheduling
- Lead management
- Call tracking
